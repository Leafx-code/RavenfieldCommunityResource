# GeometryTrack
## 描述

根据载具速度调整SkinnedMeshRenderer的混合形状权重BlendShape

## 用法

与SkinnedMeshRenderer同级（本组件已自带）

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| speedSampleTransform | Transform | 速度采样器（可选） |  
| linkDistance  | float |  |  
| vehicleRigidbody | Rigidbody | 目标载具剛体 |
