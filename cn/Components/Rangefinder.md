# Rangefinder
## 描述

用于检测物体本身至对面的距离

## 用法

挂载武器内的空物体，使空物体正对前方

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| rangeText | Text | 显示距离的Text物体（纯数字，会覆盖原文本） |  
| noReadingText  | string | 超出最大检测距离时显示的文本 |  
| maxDistance  | float | 最大检测距离 |  
| samplesPerSecond  | float | 每秒检测速度（每秒检测多少次） |
