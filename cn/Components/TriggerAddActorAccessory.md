# TriggerAddActorAccessory
## 描述
从[TriggerReceiver](./TriggerReceiver.md)类派生

官方特别行动模式中的通讯兵身上使用了本组件相同的方法

## 用法

挂载在任意空物体上即可

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| Type | enum | 添加人员装饰物或清理人员装饰物 |  
| actor | ActorReference | 要添加或清理装饰物的人员 |  
| accessory | ActorAccessory | 要添加的装饰物模型和对应的材质组 |
