# ParticlesAmbientColor
## 描述

粒子环境颜色，根据地图环境氛围颜色与日照适配粒子颜色

## 用法

与粒子系统组件同级

已自带粒子系统组件

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| weight  | float | 适配程度 |  
| ambientIntensity  | float | 环境氛围对适配颜色的权重 |  
| sunlightIntensity  | float | 日照对适配颜色的权重 |  
| useSunlightRaycast  | bool | 是否根据日照适配 |
