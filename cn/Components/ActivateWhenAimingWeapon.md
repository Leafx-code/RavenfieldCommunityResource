# ActivateWhenAimingWeapon
## 描述

当指定的weapon项处于瞄准状态时，激活指定物体

## 用法

随便挂载到任意物体

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| weapon | [Weapon](./Weapon.md) | 指定的weapon |  
| invertActivation | bool | 是否反向激活（不处于瞄准状态时，激活指定物体） |  
| objectToActivate | GameObject | 要激活物体 |
