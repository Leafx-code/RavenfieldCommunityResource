# WireGuidedMissile
## 描述
从[Rocket](./Rocket.md)类派生

由枪口引导弹道的导弹Projectile，如原版的AT反坦克导弹
## 用法

同[Projectile](./Projectile.md)，挂载到子弹物体

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| correctionGain  | float |  |  
| maxCorrection  | float |  |  
| controlLossDistance  | float | 最大引导范围（子弹飞出这个范围后不受再受控制） |
