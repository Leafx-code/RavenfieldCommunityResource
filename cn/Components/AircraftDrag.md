# AircraftDrag

## 描述

飞机阻力

## 用法

挂载在飞机刚体一栏的任意位置

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| forwardDragGain | float | 正向阻力增益 |
| sideDragGain | float | 侧面阻力增益 |
| upDragGain | float | 上升阻力增益 |
| forwardAngularDragGain | float | 向前角阻力增益 |
| sideAngularDragGain | float | 侧面角阻力增益 |
| upAngularDragGain | float | 上角阻力增益 |
