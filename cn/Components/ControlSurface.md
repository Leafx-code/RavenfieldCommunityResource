# ControlSurface
## 描述

用于根据玩家操作，控制一个飞机的机翼或舵面物体的旋转（在原旋转的基础上）

## 用法

挂载在目标效果物体

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| seat | Seat | 要监听的座位 |  
| input | Vector4 |  |  
| rotation | Vector3 | 各个轴向的旋转角度 |  
| turnSpeed  | float | 旋转速度 |
