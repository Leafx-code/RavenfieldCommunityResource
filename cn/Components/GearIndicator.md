# GearIndicator
## 描述

[ArcadeCar](./ArcadeCar.md)载具的移动指示器

## 用法

与Text物体挂载在同一个物体

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| car | [ArcadeCar](./ArcadeCar.md) | 要监听的ArcadeCar |  
| neutral  | string | 停车时显示的文本 |  
| forward  | string | 载具前进时显示的文本 |  
| reverse  | string | 载具倒车时显示的文本 |
