# PlayerRecoilCamera
## 描述

玩家相机震动，当载具收到命中的炮弹爆炸伤害时，给玩家的Camera的载具UI视角摇晃效果

## 用法

与Camera组件同级

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| activePlayerSeat | [Seat](./Seat.md) | 目标玩家座位 |  
| cameraHud | RectTransform | 载具HUB物体 |  
| weight  | float | 摇晃强度 |  
| zAxis  | Vector2 | 摇晃速度 |
