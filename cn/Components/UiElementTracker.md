# UiElementTracker
## 描述

使挂载该组件的UI元素始终跟踪指定的目标，并当`target`物体位于玩家视角前方时激活`activateWhenInFrontOfCamera`的物体

## 用法

挂载在Canvas画布下的一个UI物体

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| target | Transform | 目标物体 |  
| activateWhenInFrontOfCamera | GameObject | 要激活的物体 |
