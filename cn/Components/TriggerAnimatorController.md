# TriggerAnimatorController
## 描述
从[TriggerReceiver](./TriggerReceiver.md)类派生
## 用法

当接收到信号时，触发动画状态机对应的条件名称（triggerName）

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| animator | Animator | 要触发的动画状态机 |  
| triggerName | string | 要触发的Trigger类型的条件名称 |
