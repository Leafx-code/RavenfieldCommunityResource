# ChangeEmissionAtNight
## 描述

在夜晚改变UI物体的发光效果，使其变暗或改变颜色（类似暗黑模式）

## 用法

挂载在随意，建议挂载在UI父物体顶层

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| uiTargets | Graphic[] | 要变化的UI物体 |  
| targets | MaterialTarget[] | UI物体所用材质 |  
| multiplyEmissionColor  | Color | 颜色 |
