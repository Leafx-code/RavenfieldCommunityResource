# Rotor
## 描述

飞机螺旋桨

## 用法

套用在螺旋桨对应的父物体

具体参考官方Prefab

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| rotationSpeed | Vector3 | 各个坐标轴的旋转速度 |  
| solidRotorRenderer | Renderer | 主模型Renderer |  
| blurredRotorRenderer | Renderer | 螺旋桨阴影Renderer |  
| renderSolidRendererShadow  | bool | 是否渲染阴影 |
