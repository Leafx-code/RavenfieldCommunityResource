# ResupplyCrate
## 描述

补给箱，给range范围内的玩家与Bot补充弹药和生命值（生成后随机等待1至3秒后补给一次，之后每隔3秒补给一次）

## 用法

套用在任意空物体上即可正常使用，可以包括模型

或者直接使用`RFTools\Prefabs\Map Elements\Must Have\`内的`Supply Crate.prefab`(推荐)

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| range  | float | 触发补给的范围 |
