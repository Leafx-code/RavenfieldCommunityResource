# DeltaRotationIndicator
## 描述

装甲载具炮塔的方向指示器

使挂载这个组件的物体依照目标物体的旋转而旋转

## 用法

挂载在要旋转的指示性物体上（如UI物体）

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| target | Transform | 目标物体（如炮塔） |  
| relativeTo | Transform | 相对物体（如炮塔外的指向正前方的物体物体） | 
| targetRotationAxis | Vector3 | 目标物体的旋转矫正 |
| indicatorRotationAxis | Vector3 | 相对物体的旋转矫正 |
