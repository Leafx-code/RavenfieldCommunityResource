# SpinWhenBurning

## 描述

飞机坠机时的旋转

## 用法

参考[Vehicle](/cn/Components/Vehicle.md)

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| minTorque | Vector3 | 最小扭矩 |
| maxTorque | Vector3 | 最大扭矩 |
| allowFlipX | bool | 允许物体X轴旋转 |
| allowFlipY | bool | 允许物体Y轴旋转 |
| allowFlipZ | bool | 允许物体Z轴旋转 |
