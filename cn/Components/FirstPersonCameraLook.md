# FirstPersonCameraLook
## 描述

第一人称自由视角，允许相机随操作转动

## 用法

挂载到对应相机

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| maxHeadBobOffset | float | 摄像机晃动效果的最大偏移量 |  
| headBobPositionWeight  | float | 玩家视角操作晃动效果的权重 |  
| headBobRecoilKickWeight  | float | 武器后坐力效果对摄像机旋转角度的影响权重 |
