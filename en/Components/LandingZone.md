# LandingZone
## Description

Definition of mooring areas for boats

## Usage

Applied to an empty object

The volume of the object is the range

No other add-ons needed (like Collision, Rigidbody, etc.)

## Comment

## Variables
| Name | Type | Description |
| ----------- | ----------- | ----------- |
| target | [SpawnPoint](./SpawnPoint.md) | Position corresponding to this mooring area |
