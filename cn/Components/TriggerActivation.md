# TriggerActivation
## 描述
从[TriggerReceiver](./TriggerReceiver.md)类派生
## 用法

一般用于激活或隐藏一些特定的游戏物体

## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| objectsToActivate | GameObject[] | 接收到信号时激活该游戏物体组 |  
| objectsToDeactivate | GameObject[] | 接收到信号时隐藏该游戏物体组 |
